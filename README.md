# Pedestrian Evacuation Simulator

**Overview**  
This is a school project developed by me and 3 other students. The simulator models pedestrian movement during emergency evacuations to help analyze crowd behavior and identify potential bottlenecks in buildings or public spaces.

**Purpose**  
The project was created to study and simulate crowd dynamics, visualize evacuation processes, and evaluate different safety strategies in a controlled, interactive environment.

**Technologies**  
- **Language:** Java  
- **GUI:** JavaFX  
- **Features:** Real-time visualization, agent-based movement using the magnetic force method and evacuation metrics.
